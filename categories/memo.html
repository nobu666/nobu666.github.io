manを実行すると "WARNING: old character encoding and/or character set" とエラーを吐く時の対処

Posted on Mar 11, 2009
manを実行すると以下のようなエラーになることがある。 XXX XXX WARNING: old character encoding and/or character set XXX いろいろ調べた結果、libiconvが悪さしていた。/usr/bin/iconvではなく、/usr/local/bin/iconvを見に行っていたのが原因だった。/usr/local/bin/iconvのファイル名を変更したら治った。
