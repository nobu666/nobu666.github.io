俺メモ : bashでCSVを扱う

Posted on Jul 20, 2006
bashで本格的なCSVを扱うにはPerlなんかを使うほか道はないっぽいけど、ダブルクォート内のカンマは文字列として扱うとか、そういうことを無視して極単純なCSVを扱うだけならbashだけで以下のようにいける。 $ cat read_csv IFS=, while read FILE do set – $FILE echo “field1:” $1 “,field2:” $2 done $ cat data.csv aiueo,12345 hogehoge,mogemoge $ sh ./read_csv < ./data.csv field1:aiueo,field2:12345 field1:hogehoge,field2:mogemoge なんかPerlとか使うと負けた気がするのよね…
